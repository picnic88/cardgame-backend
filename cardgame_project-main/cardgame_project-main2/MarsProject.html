게임 설명

게임 방식

시작 화면에서 닉네임을 입력하고 게임 시작 버튼을 누르면 게임이 시작되며
1단계부터 5단계까지 구성되어 있으며 단계별 시간 제한이 다릅니다.

1단계 : 20초
2단계 : 30초
3단계 : 40초
4단계 : 50초
5단계 : 60초

점수 획득 방식

카드 한 쌍을 맞추면 +55점 획득!
단계 클리어하면 +550점 획득!

카드 한 쌍을 맞춘 경우 ‘55’점을 획득 할 수 있으며,
각 단계마다 클리어한 경우 550점을 획득 할 수 있습니다.

랭킹

랭킹은 총 30위까지 저장되며 8개씩 확인 할 수 있습니다.
또한 순위권에 해당하는 경우에는 게임 도중 실시간으로 자신의 순위를 확인 할 수 있지만,
그렇지 않은 경우에는 ‘-’로 표시됩니다.

아이템 카드

아이템 1 - 시간 추가 : 제한 시간을 10초 추가하는 기능
아이템2 - 랜덤 매칭 : 랜덤으로 카드 한 쌍을 맞춰주는 기능
아이템3 - 전체 보기 : 카드 정보 전체를 3초 동안 볼 수 있는 기능
(전체 보기 아이템은 아이템 보관함에 중복 저장이 불가하며 맨 아래칸에 자동 저장된다.)

**아이템 카드는 라운드를 넘어가도 아이템 보관함에 저장된 아이템 카드는 유지된다.
